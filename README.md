# Wave-Run
A 2D side scroller running game, with a pirate theme. The aim of the game is to outrun the incoming Tsunami wave, while collecting money, power-ups and avoiding obstacles and enemies.
## Platform Support
- Windows XP onwards
- Linux
## Requirements
- Python 3.x
- Pygame 1.9.6 onward
